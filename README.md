# Kosár

### Új termék hozzáadása

##### A termekek.json file item mintája:

`"id":` - A termék sorszáma

`"nev:"` - A termék neve

`"ar":` - A termék ára

`"img"` - A képfile neve, a képet az `img` mappában kell elhelyezni | Névadás a képnek: A termék neve kisbetűvel, egyben és ékezet nélkül

```json
{
	"id": 999,
	"nev": "Példa Név",
	"ar":  9999,
	"img": "peldanev.png"
}
```
